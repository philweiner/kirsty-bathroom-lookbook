Notes for Tiler

Dormido tile orientation: HORIZONTAL — 1200mm runs left to right, 600mm top to bottom. All repairs must match this orientation.

Sagome tile pattern: Each box contains multiple 3D designs. Dry-lay tiles before fixing to plan pattern distribution. Aim for a balanced, organic mix — avoid clustering identical patterns. See product photos for reference.

Grout joints: 2mm for Sagome tiles (Kerakoll Fugabella Color No 20, warm ivory). Mapei Ultracolor Plus Anthracite for any Dormido grout repairs.

Copper trim: 10mm straight edge. Install at the 1246mm line on Wall C and on all 4 external corners of the window reveals. Mitre or butt joints at corners — assess on site.

Wall C tile start: Begin from the 1246mm copper trim line and work DOWN with full tiles. The bottom cut row (64mm) is at floor level and will be hidden by the vinyl flooring edge.

Wall C horizontal: Centre 8 full columns. This gives 126mm cuts on each side — symmetrical and substantial enough to look intentional.

Sequence: Complete ALL wall tiling before vinyl flooring goes down. Core drill for basin waste pipe can go through tiles if needed (diamond bit) — ideally know vanity position first.
